## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`

`CLERK_SECRET_KEY`

`DATABASE_URL`

`WEBHOOK_SECRET`

`NODE_ENV`

`RAZORPAY_KEY_ID`

`RAZORPAY_KEY_SECRET`

`NEXT_PUBLIC_RAZORPAY_KEY_ID`

`GOOGLE_GENAI_API_KEY`
